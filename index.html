必要なもの

  • BOSS RC-505mkII
  • パソコン (Windows / Mac)
  • USBケーブル (A-B)

設定手順

  1. パソコンとRC-505mkIIをUSB接続
  2. パソコンの音声出力を「RC-505mkII」に設定
  3. RC-505mkII: [MENU] → USB → ROUTING を「LOOP IN」に
  4. このアプリで再生ボタンを押して音を鳴らす

問題がある場合

  • 音が出ない → 出力デバイスとUSB ROUTINGを確認
  • 音が歪む → INPUT LEVELまたは音量を下げる
